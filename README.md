#  Personal Carbon Footprint App

A full-stack web application that helps users calculate, track, and reduce their personal carbon footprint based on daily lifestyle activities.



##  Features
- User authentication using JWT
- Carbon footprint calculation (electricity, transport, fuel, etc.)
- Daily / monthly emission tracking
- Dashboard with emission history
- Personalized insights to reduce carbon emissions



##  Tech Stack
**Frontend**
- React.js
- HTML, CSS

**Backend**
- Java
- Spring Boot
- Spring Security
- Spring Data JPA

**Database**
- MySQL

**Authentication**
- JWT (JSON Web Tokens)
